# Data Science Articles and Technical Notes

This repository contains the computational material supporting my data science articles and technical posts. These articles are typically published on platforms such as Medium or other technical outlets, but the repository itself serves as the primary location for the accompanying code and reproducible analyses.

The goal of this repository is to provide transparent access to the computational work behind each article. Rather than presenting isolated snippets of code, the materials included here reflect complete analytical workflows that allow readers to explore the methods, experiments, and modeling decisions discussed in the written content.

The repository includes computational artifacts implemented primarily in **Python** and **R**, such as:

- **Jupyter notebooks** implementing models, experiments, and exploratory analyses  
- **R scripts** supporting statistical modeling, visualization, and complementary implementations  

These materials accompany the written articles and provide additional technical detail, enabling readers to reproduce results, inspect modeling choices, and explore the underlying computational experiments.

This repository therefore functions as a companion to my technical writing on data science, machine learning, and quantitative modeling. Many of the examples draw on my background in physics, applied mathematics, and machine learning, combining theoretical reasoning with practical experimentation.
